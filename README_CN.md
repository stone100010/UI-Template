# 🎨 AI UI 设计生成器

一个现代化的AI UI设计生成器，包含多种设计风格的HTML模板和智能预览工具。

## ✨ 项目特色

- 🤖 **AI智能生成** - 基于需求描述自动生成UI设计
- 🎯 **多风格模板** - 9种精心设计的UI风格模板
- 👀 **实时预览** - 内置AI设计预览器
- 📱 **响应式设计** - 适配各种设备屏幕
- 🚀 **开箱即用** - 完整的HTML/CSS/JS代码
- 🎨 **现代设计** - 基于Tailwind CSS和FontAwesome

## 🎨 设计风格

### 🌌 赛博朋克风格 (Cyberpunk)
- **配色方案**: 深色背景 + 霓虹色彩
- **设计元素**: 霓虹发光、故障艺术、科技感
- **适用场景**: 游戏、科技应用、未来感产品

### 💫 现代简约风格 (Modern-Minimal)
- **配色方案**: 简洁白色 + 柔和色彩
- **设计元素**: 扁平化、极简线条、留白设计
- **适用场景**: 企业应用、工具软件、效率应用

### 🌿 自然有机风格 (Organic-Natural)
- **配色方案**: 绿色系 + 自然色调
- **设计元素**: 圆润形状、自然纹理、生物形态
- **适用场景**: 健康应用、环保产品、生活类应用

### 📻 复古经典风格 (Retro-Vintage)
- **配色方案**: 暖色调 + 经典色彩
- **设计元素**: 复古字体、纹理效果、怀旧元素
- **适用场景**: 品牌官网、艺术展示、创意作品

### 🌐 3D立体风格 (3D)
- **配色方案**: 立体感配色 + 光影效果
- **设计元素**: 3D视觉效果、深度感、空间层次
- **适用场景**: 游戏界面、产品展示、互动应用

### 🎨 包豪斯设计风格 (Bauhaus)
- **配色方案**: 原色系 + 功能性配色
- **设计元素**: 几何形状、简洁线条、模块化设计
- **适用场景**: 教育应用、企业网站、设计系统

### 📱 扁平化设计风格 (Flat)
- **配色方案**: 高饱和度色彩 + 简洁配色
- **设计元素**: 扁平化图标、简洁按钮、清晰排版
- **适用场景**: 移动应用、简洁工具、快速原型

### 🚀 未来主义风格 (Futurism)
- **配色方案**: 冷色调 + 科技蓝色系
- **设计元素**: 流线型设计、渐变效果、科技纹理
- **适用场景**: 科技产品、创新应用、未来概念

### ⚙️ 工业风格 (Industrial)
- **配色方案**: 工业金属色 + 暗色调
- **设计元素**: 金属质感、网格图案、机械元素
- **适用场景**: 工业软件、仪表盘、工具应用

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）

```bash
# 给脚本添加执行权限
chmod +x start.sh

# 启动项目
./start.sh
```

访问地址：
- **本机访问**: http://localhost:8000/index.html
- **局域网访问**: http://[你的IP地址]:8000/index.html

### 方法二：手动启动

```bash
# 启动Python HTTP服务器
python3 -m http.server 8000 --bind 0.0.0.0 --directory .

# 然后访问
http://localhost:8000/index.html
```

## 📋 使用说明

### AI设计生成器功能
- **智能输入**: 输入需求描述，AI将生成对应的UI设计
- **多风格展示**: 预设9种设计风格的模板供选择
- **实时预览**: 选择设计风格即可实时预览效果
- **卡片式浏览**: 直观的设计风格展示界面
- **全屏预览**: 点击卡片进入全屏预览模式
- **新窗口打开**: 在新标签页中查看完整设计
- **响应式设计**: 自动适配不同设备的显示效果

### 自定义模板
1. 在 `htmls/` 目录下创建新的HTML文件
2. 遵循项目的设计规范和代码结构
3. 使用Tailwind CSS进行样式设计
4. 更新 `index.html` 中的 `htmlFiles` 数组，添加新模板信息

## 🔧 自定义开发

### 添加新的UI模板
1. 在 `htmls/` 目录创建新的HTML文件
2. 参考现有的模板结构和设计风格
3. 在 `index.html` 的JavaScript中更新 `htmlFiles` 数组
4. 添加文件名、显示名称、描述等信息

### 修改AI设计生成器
- 编辑 `index.html` 中的JavaScript代码
- 自定义样式和交互效果
- 扩展AI生成功能的提示词配置
- 添加新的UI设计功能特性

### 更新提示词配置
在 `index.html` 的JavaScript中，你可以自定义：
- 设计风格描述的提示词
- 不同风格模板的映射逻辑
- 生成过程中的用户交互
